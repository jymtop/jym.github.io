# Research Group Website

This is the source code for the Research Group website, hosted on GitHub Pages.

## Project Structure

- `index.html`: Home page (Recent Research Progress)
- `members.html`: Team Members
- `research.html`: Work Outcomes (Publications, Projects, Patents)
- `news.html`: Full News List
- `style.css`: Global Stylesheet

## Development

This project uses "Vanilla Web Development" style (HTML5 + CSS3), with no build tools or frameworks required.

To update content, simply edit the corresponding `.html` files.
